Install VMware Workstation in Ubuntu 14.04

2016-04-21 Thu    by Justin C.    Category: Tools    Tags: VMware, Ubuntu, 


1. Use apt-get update system: 

linuxidc@localhost:~$ sudo apt-get update
linuxidc@localhost:~$ sudo apt-get upgrade



2. Download VMware Workstation bundle.

3. Make bundle executable: 

linuxidc@localhost:~$ chmod a+x VMware-Workstation-Full-11.0.0-2305329.x86_64.bundle



4. Run the bundle: 

linuxidc@localhost:~$ sudo ./VMware-Workstation-Full-11.0.0-2305329.x86_64.bundle
